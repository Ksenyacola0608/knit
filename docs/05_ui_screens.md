# UI Экраны: Платформа услуг ручного труда

## 1. Главная страница (Home)

### 1.1 Компоненты
- **Hero Section**
  - Заголовок: "Найдите мастера ручного труда"
  - Подзаголовок: "Уникальные изделия от талантливых мастеров"
  - Поисковая строка с категориями
  - CTA кнопка "Начать поиск"

- **Популярные категории**
  - Grid карточек категорий (Вязание, Вышивка, Шитье и т.д.)
  - Иконки для каждой категории
  - Количество услуг в категории

- **Популярные мастера**
  - Slider с карточками мастеров
  - Аватар, имя, рейтинг, специализация

- **Как это работает**
  - 3 шага: Найти → Заказать → Получить
  - Иконки и краткое описание

## 2. Страница каталога услуг (Services)

### 2.1 Layout
```
┌────────────────────────────────────────────────────────────┐
│  Header + Navigation                                       │
├──────────┬─────────────────────────────────────────────────┤
│          │  [Поиск]                    [Сортировка ▼]      │
│ Фильтры  ├─────────────────────────────────────────────────┤
│          │  ┌────────┐  ┌────────┐  ┌────────┐            │
│ □ Катего │  │Service │  │Service │  │Service │            │
│ □ Цена   │  │ Card 1 │  │ Card 2 │  │ Card 3 │            │
│ □ Рейтинг│  └────────┘  └────────┘  └────────┘            │
│          │  ┌────────┐  ┌────────┐  ┌────────┐            │
│          │  │Service │  │Service │  │Service │            │
│          │  │ Card 4 │  │ Card 5 │  │ Card 6 │            │
│          │  └────────┘  └────────┘  └────────┘            │
│          │                                                  │
│          │  [← Previous] [1] [2] [3] [Next →]              │
└──────────┴─────────────────────────────────────────────────┘
```

### 2.2 Service Card
- Изображение услуги
- Название услуги
- Краткое описание (первые 100 символов)
- Цена
- Информация о мастере (аватар, имя, рейтинг)
- Кнопка "Подробнее"

### 2.3 Фильтры
- **Категории**: чекбоксы
- **Цена**: слайдер (мин-макс)
- **Рейтинг мастера**: от 1 до 5 звезд
- Кнопка "Применить" и "Сбросить"

### 2.4 Сортировка
- По дате (новые)
- По рейтингу (высокие)
- По цене (возрастание/убывание)

## 11. Модальные окна

### 11.1 Order Form Modal (Форма заказа)
- Название услуги (readonly, с изображением)
- Описание заказа (textarea)
- Пожелания (textarea)
- Загрузка файлов (опционально)
- Кнопки: "Отправить заказ" / "Отмена"

### 11.2 Review Form Modal (Форма отзыва)
- Название заказа (readonly)
- Рейтинг (звездочки 1-5)
- Комментарий (textarea)
- Кнопки: "Отправить" / "Отмена"

### 11.3 Service Form Modal (Форма создания/редактирования услуги)
- Название
- Описание
- Категория (select)
- Цена
- Срок выполнения (дни)
- Загрузка изображений
- Кнопки: "Сохранить" / "Отмена"

## 12. Компоненты UI (общие)

### 12.1 Header
- Логотип
- Навигационное меню:
  - Каталог
  - Мои заказы (для авторизованных)
  - Чат (с badge непрочитанных)
  - Уведомления (с badge)
  - Профиль (dropdown: Настройки / Выйти)

### 12.2 Footer
- Ссылки на разделы
- Социальные сети
- Копирайт

### 12.3 Loading States
- Skeleton screens для карточек
- Spinner для кнопок
- Progress bar для загрузки изображений

### 12.4 Empty States
- "Нет услуг" в каталоге
- "Нет заказов" на странице заказов
- "Нет сообщений" в чате

### 12.5 Toast Notifications
- Success (зеленый)
- Error (красный)
- Info (синий)
- Warning (желтый)

## 13. Адаптивность

### 13.1 Mobile (< 768px)
- Hamburger menu для навигации
- Single column layout
- Фильтры в drawer/modal
- Touch-friendly кнопки (минимум 44x44px)

### 13.2 Tablet (768px - 1024px)
- 2-column grid для карточек услуг
- Sidebar фильтров скрывается в drawer

### 13.3 Desktop (> 1024px)
- 3-4 column grid
- Фиксированный sidebar фильтров
- Расширенные карточки с hover эффектами
